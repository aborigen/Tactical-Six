# Tactical Six | Advanced 6x6 Strategic Chess

Tactical Six is a sophisticated 6x6 chess variant designed for fast-paced, high-intensity strategic engagement. It features a custom-built local tactical engine and a modern, high-fidelity interface.

## 🚀 Key Features

- **6x6 Matrix Combat**: A condensed arena that forces tactical engagement and rewards precise positioning.
- **Local Tactical Engine**: Powered by a custom Minimax algorithm with Alpha-Beta pruning, providing instant strategic analysis and an AI opponent without external API calls.
- **Dynamic Visuals**: High-contrast piece sets, last-move arrow indicators, and real-time tactical overlays.
- **AI Command Center**: Integrated strategy hints and manoeuvre explanations to assist in mastering the 6x6 meta.
- **Responsive Command**: Seamlessly switch between Local PvP and VS Engine modes.

## 🛠 Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [ShadCN UI](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **State Management**: React Hooks & Custom Chess Logic

## 🎮 How to Play

1. **Deployment**: White and Black start on opposite sides of the 6x6 grid.
2. **Matrix Rules**: Standard chess rules apply, but within a reduced 6x6 space. Pieces include Rook, Knight, Bishop, Queen, King, and Pawns.
3. **Winning**: Achieve checkmate or force a stalemate through superior tactical positioning.
